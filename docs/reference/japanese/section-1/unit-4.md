# Section 1 Unit 4
## Describe Belongings
### Vocabulary
| English | Japanese |
|:-------:|:--------:|
| these | これら |
| those | それら |
| my | わたしの |
| hat | ぼうし |
| umbrella | かさ |
| shoe | くつ |
| coat | コート |
| red | あかい |
| white | しろい |

### Examples
| English | Japanese |
|:-------:|:--------:|
| This is my hat. | これはわたしのぼうしです。 |
| My umbrella is red. | わたしのかさはあかいです。 |
| Where is my shoe? | わたしのくつはどこですか。 |
| These are my coats. | これらはわたしのコートです。 |
